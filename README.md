📊 Amazon Prime Video Data Analysis

🔍 Overview

This project focuses on analyzing the Amazon Prime Video dataset to uncover trends, patterns, and insights related to movies and TV shows available on the platform. The analysis helps understand content distribution, genres, ratings, and release trends.

🎯 Objective

To analyze the Amazon Prime Video dataset
To explore content trends across years, genres, and ratings
To visualize insights using Python libraries
To build a strong foundation in data analysis as a fresher

🛠️ Tools & Technologies

Python

Pandas

NumPy

Matplotlib

Seaborn

Jupyter Notebook

📂 Dataset

Dataset: Amazon Prime Video Dataset

Contains information such as:

Title

Genre

Release Year

Ratings

Duration

Type (Movie/TV Show)

🔧 Project Workflow

1. Data Cleaning & Preprocessing

Handled missing values

Removed duplicates

Converted data types for analysis

2. Exploratory Data Analysis (EDA)

Analyzed distribution of movies vs TV shows

Explored release trends over the years

Identified most popular genres

3. Data Visualization

Bar charts for genre distribution

Line charts for yearly trends

Heatmaps for correlations

📊 Key Insights

Movies dominate the platform compared to TV shows

Significant growth in content release after 2015

Certain genres like Drama and Comedy are most common

Ratings distribution shows majority content falls in mid-range

📈 Visualizations

Genre distribution charts

Year-wise content growth

Rating analysis graphs

🚀 Results


This project demonstrates how raw entertainment data can be transformed into meaningful insights using data analysis techniques. It highlights skills in data cleaning, visualization, and exploratory analysis.
